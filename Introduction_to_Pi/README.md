
# Introduction to Raspberry Pi 
This notebook is an introduction to Raspberry Pi.    

---

## Contents   
This notebook covers 
* Setting up a Raspberry Pi 
* Enabling remote-access 
* Changing Raspberry Pi password, adding and deleting users
* Changing time and date
* Installing and opening Jupyter Notebook on Pi
* Setting up Pi camera
* Taking photo and video using Pi camera
* Shutdown the Raspberry Pi
